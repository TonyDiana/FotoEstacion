Montaje¶

En este apartado encontraremos las instrucciones para montar cada uno de los periféricos que se vayan incorporando al proyecto, tanto las especificaciones del hardware que hay que adquirir, como el esquema de montaje, los planos para imprimir las cajas contenedoras y todo el procedimiento para instalar el software adecuadamente.

Puesto que este es un proyecto que no está financiado ni patrocinado por ninguna entidad, no se ofrecerán enlaces a tiendas reales o virtuales sobre donde adquirir los materiales, pero sí se informará de forma suficientemente clara y concisa de los materiales empleados, motivo por el cual han sido utilizados y especificaciones necesarias para adquirirlos.

Se darán indicaciones sobre varias opciones de compra, los elementos probados, los que son posiblemente compatibles y aquellos que serían posibles pero que requieren pruebas por parte del usuario final para adaptarlos.

Atención

No se ofrece ninguna garantía ni soporte sobre este software ni el manejo o montaje de hardware. En caso de seguir las instrucciones, instalarlo y mucho más en el caso de realizar adaptaciones personalizadas, el usuario acepta esta solución libre de cargas, pero sin derechos a reclamos, indemnizaciones o ayuda por parte del equipo desarrollador.

Por otro lado, no se ha ofrecido ninguna información errónea ni se se han omitido detalles, salvo errores involuntarios, sobre cómo debe montarse o utilizarse esta solución.
