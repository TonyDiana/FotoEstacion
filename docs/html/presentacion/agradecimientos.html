Agradecimientos¶

Este proyecto ha sido posible gracias a varias colaboraciones externas; resulta más que conveniente y justo nombrar dichos aportes de la manera adecuada, pero antes de nada, recuerda que:

¡¡ ESTE PRODUCTO NO ESTÁ PATROCINADO ECONÓMICAMENTE POR NINGUNA ENTIDAD !!

A nivel general¶

  • El primer agradecimiento se lo debemos al autor filosófico de la idea: Tony Diana.
  • Los iconos de los temas oficiales han sido obtenidos de MaterialDesign.

Librerías de terceros de Python¶

  • La base de datos interna ha sido desarrollada con TinyDB.
  • Se utilizó la librería Simple Python classproperty decorator para algunas de las clases.

De parte de HelioIsis¶

  • Algunos de los elementos utilizados son sensores fabricados por la compañía Adafruit. El aporte libre de sus elementos y librerías de manejo fué muy importante para lograr desarrollar este producto.
  • El moderador del foro de Arduino en español, @Surbyte ha sido una de las personas más importantes para poder poner en funcionamiento este proyecto. Incluso cuando el proyecto migró de Arduino a Linux continuó ofreciendo su valiosa ayuda y experiencia.
